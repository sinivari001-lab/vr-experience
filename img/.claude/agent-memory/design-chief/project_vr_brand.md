Brand identity v1.0 delivered on 2026-04-06 for VR Experience.

**Why:** Company needed full visual identity for website integration, physical brindes (camisetas, canecas, adesivos, chaveiros, banners, cartoes de visita), and brand consistency across digital and print.

**How to apply:** All brand assets live at `sites/vr-experience/brand/`. The logo-inline.svg is designed for navbar replacement on index.html. Favicon.svg replaces the emoji favicon. Brand guide contains print production specs (Pantone references, CMYK, bleed, paper stocks) ready for real graficas.

Key decisions:
- Headset icon with dual-lens design (cyan left, magenta right) as core brand mark
- Orbitron 800 for "VR", Orbitron 400 for "EXPERIENCE" creates visual hierarchy
- Light background versions use darkened neon tones (#0099aa, #cc00cc) instead of pure neons
- Monochrome versions use #1a1a2e (not pure black) for softer look on light backgrounds
