## Dicas de Uso

1. **Midjourney v6**: Adicione `--no text` se a IA gerar texto incorreto. Gere a imagem sem texto e adicione em pos-producao.
2. **DALL-E 3**: Se o texto ficar errado, peca especificamente para soletrar cada palavra.
3. **Cores exatas**: Sempre mencione os hex codes (#00b8d4, #7c4dff, #0a0a1a, #00e676) nos prompts.
4. **Pos-producao**: Use os SVGs deste diretorio como referencia para sobrepor as artes nos mockups gerados.
5. **Resolucao**: Para impressao, gere em resolucao maxima e faca upscale para 300 DPI.
